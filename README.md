# luc (language universal code)

Use pyinstall to make into a exe

To just run code

`./luc -i main.luc`

To complie

`./luc -c main.luc ruby`
or
`./luc -c main.luc python`
or
`./luc -c main.luc c`
or
`./luc -c main.luc c++`

To debug

`./luc -d main.luc`

To get help

`./luc --help`

To enter the playground

`./luc -p`
